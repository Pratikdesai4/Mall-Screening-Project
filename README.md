# American Dream | Interactive Sales Experience

A high-fidelity, cinematic interactive sales deck designed for the American Dream mall in East Rutherford, NJ. This project showcases the scale, luxury, and commercial potential of one of the world's premier retail and entertainment destinations.

## 🚀 Live Demo
[Link to Vercel/GitHub Pages Deployment]

## ✨ Key Features
- **Cinematic Storytelling**: Immersive video backgrounds and scroll-triggered animations.
- **Interactive Data**: Real-time stat counters and catchment area visualization with hotspots.
- **Luxury Branding**: Dedicated "Avenue" section with premium typography and logo grids.
- **Modular Expandability**: Independent overlay modules for Events, Sponsorship, and Leasing.
- **Performance Optimized**: Lazy-loaded assets, intersection observer triggers, and responsive glassmorphism UI.

## 🛠 Tech Stack
- **Core**: Semantic HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Vanilla CSS3 with Custom Properties (CSS Variables)
- **Animations**: Intersection Observer API, CSS Transitions/Animations, requestAnimationFrame
- **Typography**: Google Fonts (Cinzel, Outfit)
- **Media**: YouTube Embed API (with fallback system), AI-generated assets

## 🧠 Design Decisions & AI Collaboration
- **Aesthetic**: A "Dark Luxury" theme using HSL-curated gold gradients (`#cba153`) and deep obsidian backgrounds (`#030303`).
- **User Flow**: A vertical "cinematic" scroll journey inspired by high-end digital agency portfolios (e.g., Active Theory, Digideck).
- **AI Integration**:
  - **Visuals**: Fallback images and conceptual section visuals were generated using AI to maintain a cohesive, futuristic aesthetic.
  - **Development**: Antigravity (AI) was used to accelerate component building, refine responsiveness, and implement complex intersection observer logic for performance.
